# ウルトラマンプロジェクト（Ultraman Project）

## プロジェクト概要
PHP と MySQL を用いて開発した掲示板型フォーラムシステム。  
ユーザーは会員登録、ログイン、投稿、返信、コメントなどの機能を利用できる。

## 使用技術

### フロントエンド
- HTML
- CSS
- JavaScript
- JSON（フロントエンドとバックエンド間のデータ通信）

### バックエンド
- PHP
- AJAX（非同期通信による機能実装）

### データベース・開発環境
- MySQL / MariaDB
- XAMPP（ローカル開発環境）
- AWS EC2（サーバー環境へのデプロイ）
- GitHub（バージョン管理）
- WinSCP / PuTTY（リモート接続・ファイル転送）
