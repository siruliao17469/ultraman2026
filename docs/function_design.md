# 機能設計書

## 1. 機能一覧

| 機能カテゴリ | 機能名 | 機能概要 |
|------------|--------|--------|
| ユーザー管理 | 会員登録・ログイン | ユーザーがアカウントを登録し、ログインできる機能 |
| ユーザー管理 | プロフィール編集 | ユーザーが自分の基本情報を編集できる機能 |
| ユーザー管理 | マイページ | ユーザーが自分の投稿、コメント、プロフィールを確認できる機能 |
| 投稿管理 | 投稿作成 | ユーザーが新しい投稿を作成できる機能 |
| 投稿管理 | 投稿編集 | ユーザーが自分の投稿を編集できる機能 |
| 投稿管理 | 投稿削除 | ユーザーが自分の投稿を削除できる機能 |
| コメント管理 | コメント投稿 | ユーザーが投稿にコメントできる機能 |
| コメント管理 | コメント返信 | ユーザーがコメントに返信できる機能 |
| コンテンツ閲覧 | 画像閲覧 | ユーザーがサイト上のウルトラマン画像を閲覧できる機能 |
| 管理者機能 | ユーザー管理 | 管理者がユーザー情報を確認・管理できる機能 |
| 管理者機能 | 投稿管理 | 管理者が投稿を編集・削除できる機能 |
| 管理者機能 | コメント管理 | 管理者がコメントを管理・削除できる機能 |

## 2. 権限区分

| 権限 | 内容 |
|------|------|
| 一般ユーザー | 投稿の作成・編集・削除、コメント、プロフィール編集が可能 |
| 管理者 | すべてのユーザー・投稿・コメントの管理が可能 |

## 3. 機能概要説明

### 3.1 ユーザー管理機能
ユーザーの登録、認証、プロフィール管理を行う機能。

### 3.2 投稿管理機能
掲示板への投稿作成、編集、削除を行う機能。

### 3.3 コメント管理機能
投稿に対するコメントおよび返信を行う機能。

### 3.4 管理者機能
管理者によるユーザー、投稿、コメントの管理を行う機能。
